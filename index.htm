As principais tags HTML


< html >
Define o início e o fim do documento HTML.


< head >
Contém informações sobre a página, como título, metadados, links para folhas de estilo e outros elementos.



< title >
Define o título da página exibido na barra de título do navegador.



< meta >
Usado para fornecer metadados sobre a página, como codificação de caracteres, descrição e palavras-chave.



< link >
Utilizado para vincular folhas de estilo externas ao documento HTML.



< style >
Permite a inclusão de estilos CSS diretamente na página.



< script >
Utilizado para incluir scripts JavaScript na página.



< body >
Contém o conteúdo visível da página, como texto, imagens, vídeos e outros elementos.



< h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 >
Tags de cabeçalho usadas para criar títulos e subtítulos com diferentes níveis de importância.



< p >
Define um parágrafo de texto.



< a >
Cria hiperlinks para outras páginas ou recursos.



< img >
Usado para incorporar imagens na página.



< ul >
Cria uma lista não ordenada.



< ol >
Cria uma lista ordenada.



< li >
Define um item em uma lista, seja ordenada ou não.



< div >
Usado para criar blocos de conteúdo ou containers que podem ser estilizados com CSS.



< span >
Define uma parte de texto em linha que pode ser estilizada com CSS.



< table >
Utilizado para criar tabelas.
